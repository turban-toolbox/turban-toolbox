# Welcome to the Turbulence analysis (TURBAN) toolbox

This is the main documentation hub for TURBAN.

* [Quickstart](quickstart.md) - Jump straight in.
* [User Manual](manual.md) - Documentation of the core (instrument-agnostic) functions.
* [Developer's Guide](devguide.md) - Technical documentation for
  developers.
* [Test and benchmark data files](benchmark_files.md) - How to get
  access to test and benchmark data files 
* [API](api.md) - The full API of turban.

In addition, TURBAN comes with a (growing) number of tools to read data from common turbulence instruments and convert them to TURBAN-compliant data structures.
Currently, there are two:
* [MSS Reference](mss.md) - Technical details of the functions and classes to process MSS data.
* [MicroRider Reference](microrider.md) - Technical details of the functions and classes to process MicroRider data.
